# BibApi

Built with [NestJS](https://nestjs.com/). It provides a REST API to manage the data of the
BibCnrs. It uses a PostgreSQL database.

## Installation

Use the Makefile commands at the root of the project to install the package.

```bash
make install
```

## Usage

```bash
make start
make logs-api
```
